Description and Features

Add functionalities to compute and display the stock rotation for the products

Module Features :

  • new report "Stock Rotation Analysis" in Reporting/Warehouse : to analyse stock rotation in a selected period
    • Internal Reference : product internal reference
    • Name : product name
    • Sold Quantity : product sold quantity in selected period
    • Average Stock : (Stock at beginning of the period + Stock at end of the period) / 2
    • Stock Rotation : Quantity of product sold / Average stock in selected period
